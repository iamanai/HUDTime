--------------------------------------------------
ABOUT

HUDTime is a Rainmeter version of the Windows 7 gadget, "HUD Time". It consists of a simple time and date display that can be placed
anywhere on the desktop.

Configuration options include: Text color, transparency, font, and 12/24 hour time.
